# 🎵 Melodia

Melodia is a beautiful music player app 🎶 built to enhance Flutter skills.

It allows you to:

- Play, pause ⏯️, skip next ⏭️,play previous ⏮️ and loop 🔁 songs
- Keep music playing even when the app is closed or the screen is off 🔒  
- Show song info in the notifications panel 🔔  
- Mark songs as favorites ❤️ which are stored in local storage using Hive 🐝  
- Switch between Light 🌞 and Dark 🌙 modes for a better user experience  

---

## 🛠️ Built With

- Flutter 💙  
- GetX ⚡  
- Hive 🐝  
- MVC Architecture 🧱
